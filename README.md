# Full Time CEO Apparel

A single-page HTML storefront for **Full Time CEO Apparel** — no build tools, no dependencies, no framework. Just open `index.html` in a browser.

## Live Preview

Deploy instantly via [GitHub Pages](https://pages.github.com/):
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to the `main` branch, root `/`
4. Your site will be live at `https://<username>.github.io/<repo>/index.html`

## File Structure

```
.
├── index.html   # Complete self-contained storefront
├── README.md
└── .gitignore
```

## Stack

- Plain HTML5 / CSS3 / Vanilla JS — zero dependencies
- Google Fonts (Bebas Neue, Barlow Condensed, Barlow) loaded via CDN
- All styles and scripts are inline — one file, fully portable

## Getting Started

```bash
git clone https://github.com/<username>/<repo>.git
cd <repo>
open index.html   # macOS
# or: start index.html  (Windows)
# or: xdg-open index.html  (Linux)
```
